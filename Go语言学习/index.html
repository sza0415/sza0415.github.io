引用类型（reference type）特指slice、map、channel

new 函数

new是go语言的内置函数，用于为制定类型分配内存，并返回一个指向该类型零值的指针。

  • 适用于基本数据类型（如int、float64、bool等）以及结构体（struct）等类型。
  • 不适用于切片（slice）、映射（map）、和通道（channel）引用类型的初始化，因为new仅分配内存并返回指向零值的指针，不会初始化这些类型的内部数据结构。
  • 返回值：指向所分配类型的指针，例如：*int、*struct
make 函数

make是 Go 语言的内置函数，专门用于初始化并返回切片（slice）、映射（map）和通道（channel）这三种引用类型。它不仅分配内存，还会初始化这些类型的内部数据结构，使其可以直接使用。

  • 仅适用于切片、映射和通道。

  • 不适用于基本数据类型或结构体。

  • 返回值：返回可用的引用类型实例，例如：[]int、map[string]int、chan int

指针

不能将内存地址和指针混为一谈。

内存地址是内存中每个字节单元的唯一编号，而指针是一个实体，指针会分配内存空间，相当于一个专门用来保存地址的整形变量。

在 Go 语言中，不能直接获取 map 中元素的地址，因为 map 的底层实现是哈希表，元素的存储位置会随着 map 的增长、删除等操作动态变化。获取地址没有固定的意义，因此 Go 语言禁止了这种操作。

switch

无需显示执行break语句，case执行完毕后自动中断。如需执行后续case（源码顺序），需执行fall through，但不再匹配后续条件表达式。

for .. range

for .. range 数据迭代，支持字符串、数组、数组指针、切片、字典、通道类型，返回索引、键值数据。

  • string：索引 (index) 和 Unicode 码点 (rune)
    • 类型为 rune，实际上是 int32 的别名）。如果你想将 r（即 Unicode 码点）转换回对应的字符，可以使用 string() 函数
  • array：索引 (index) 和元素值 (value)
  • array Pointer：同数组，索引 (index) 和元素值 (value)
  • slice：索引 (index) 和元素值 (value)
  • map：键（key）和值（value）
    • 注意: 遍历顺序是随机的
  • channel：只返回通道中的值 (value)，没有索引
    • 注意: 循环会持续读取通道中的数据，直到通道关闭

注意：range会复制目标数据。
